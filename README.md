# 100 Days of Code Python

This repository tracks my progress through Python fundamentals, problem solving, and beginner automation projects.

## Purpose

The goal of this repo is to build consistency with Python while improving my understanding of variables, conditionals, loops, functions, lists, dictionaries, file handling, APIs, and automation.

## Project Structure

Each project is organized into its own folder. Every folder includes a main.py file and a short README explaining the project.
