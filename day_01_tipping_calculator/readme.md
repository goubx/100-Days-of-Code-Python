# Tipping Calculator

## Project Overview

This project calculates how much each person should pay when splitting a bill with a tip included.

## Concepts Practiced

Input  
Variables  
Arithmetic  
Float conversion  
Rounding  

## What the Program Does

The program asks for the total bill, tip percentage, and number of people splitting the bill. It then calculates the final amount each person should pay.

## What I Learned

I practiced using user input, converting values, performing math operations, and formatting the final result.
